## Spring Boot SSL

### Technologies
Spring Boot, SSL


### Steps
##### Build and Run Application
via spring-boot-maven-plugin: <br />
*mvn spring-boot:run*

##### Build jar, Run jar
*mvn package* <br />
*java -jar target/jar-name.jar*

##### Access Application
*https://localhost:8443*
